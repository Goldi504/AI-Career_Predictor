# 🚀 AI Career Predictor & Skill Gap Analyzer

![MERN Stack](https://img.shields.io/badge/MERN-Stack-green)
![AI Powered](https://img.shields.io/badge/AI-Powered-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ⚙️ How It Works

1. User registers and logs in.
2. Completes profile and adds skills.
3. AI analyzes user skills and interests.
4. Suitable career paths are predicted.
5. Skill gaps are identified.
6. Personalized recommendations are generated.
7. AI creates a learning roadmap.
8. User tracks progress through the dashboard.
9. Skills are updated as learning progresses.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Goldi504/AI-Career_Predictor.git
cd AI-Career_Predictor
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend directory:

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

GEMINI_API_KEY=your_api_key
```
